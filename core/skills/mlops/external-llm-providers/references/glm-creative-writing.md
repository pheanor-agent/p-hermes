# GLM 모델 창의 작업 성능 비교

**세션**: 2026-05-22 (페아노르 요청)
**테스트 대상**: 마이너 웹소설 제목 5개 생성

## 결과 요약

| 모델 | 응답 시간 | 스타일 | 품질 |
|------|----------|--------|------|
| Qwen3.6 | ~10초 | 일반적 | 보통 |
| GLM-4.7 | ~30초 | 감성적, 관념적 | 좋음 |
| GLM-5-turbo | ~90초 | 길고 구체적, "[ ]" 스타일 | 좋음 |
| GLM-5.1 | 3분+ | 장르별 균형, 직관적 | 좋음 |

## 사용자 평가

> "소설에 필요한 창의성은 qwen3.6 같은 소규모 모델보다 glm 5 turbo 같은 대규모 모델이 더 유리한 것 같아"

## 모델별 스타일 특징

### GLM-4.7
- 감성적이고 관념적인 제목 선호
- 예: "꺼진 전구의 소리", "유령 먹는 법", "방 안의 코끼리가 깊은 숨을 쉬다"
- 분위기 중심, 문학적

### GLM-5-turbo
- 길고 구체적인 문장형 제목
- "[ ]" 또는 " "로 감싸는 스타일
- 예: "9번째 회귀자가 이번에는 그냥 쓰러져서 자기로 했습니다"
- 마이너 웹소설 특유의 피로감/엇박자 강조

### GLM-5.1
- 장르별 균형 잡힌 접근 (무협/공포/직업물/일상물/대체역사)
- 직관적이고 스토리텔링 중심
- 예: "정파를 관둔 노련한 검객, 마교의 인사과에 이력서를 냈습니다"

## 실행 팁

- 창의 작업: `glm-4.7` (빠름+질량 충분) 또는 `glm-5-turbo` (깊이)
- `max_tokens`: 2000-8000 (reasoning 모델은 5000+ 권장)
- 한국어 요청: `--max-time 60` 이상 설정
