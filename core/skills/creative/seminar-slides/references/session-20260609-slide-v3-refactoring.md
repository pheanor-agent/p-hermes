# JOB-1555: 슬라이드 v3.0 대대적 리팩토링

## 세션 요약

사용자가 슬라이드 v2.1 에 대한 종합적인 피드백을 제공했고, v3.0 으로 대대적으로 리팩토링했습니다.

## 주요 변경 사항

### 내용 수정
- **S3**: 백과사전→학교 교육, 참고 자료→매뉴얼&위키, 순서 교체 (2↔3)
- **S4**: `two-col` 레이아웃 → 단일 컬럼 통합, 비교 설명 추가
- **S6**: 한문 `和` 제거 ("기존知识和" → "기존 지식과")
- **S7**: CronJob/Script/Pipeline → 자동 스케줄러/작업 명령어/3 단계 연결
- **S8**: Tier 참조 방식 팝업 추가
- **S9/S10**: 목적 설명 Tip 배지 보완

### 기능 추가
- 이미지 확대/축소/이동 기능 (모달 오버레이)
- 마우스 휠 줌, 드래그 이동, ESC 닫기

### 버그 수정
- `<style>` 태그 분할 버그 (patch 도구로 CSS 추가 시 중간에 닫힘)
- JavaScript 전역 스코프 문제 (DOMContentLoaded 내부 함수 정의 시 onclick 접근 불가)
- JSON 따옴표 충돌 (이중 따옴표 → 단일 따옴표)

## 발견된 패턴

### 1. CSS 태그 분할 버그
`patch` 도구로 CSS 블록을 추가할 때, `old_string` 이 `</style>` 을 포함하지 않으면 새로운 CSS 가 닫히지 않은 상태로 삽입됩니다. 다음 수정에서 `</style>` 이 중복되거나 누락될 수 있습니다.

**해결**: patch 후 `grep -n '</style>'` 으로 중복 확인.

### 2. JavaScript 인라인 onclick 스코프
HTML 인라인 `onclick="nextSlide()"` 은 전역 스코프에서 함수를 찾습니다. `DOMContentLoaded` 내부에 정의된 함수는 접근할 수 없습니다.

**해결**: 함수를 전역 스코프에 정의하거나 `addEventListener` 사용.

### 3. 한문 자동 검증
Chinese characters can sneak in during AI-generated content. Use `grep -E "[一-龺]|[ぁ-ん]|[あ-ん]" index.html` to detect.
