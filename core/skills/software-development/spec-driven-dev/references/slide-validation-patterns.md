# 슬라이드 검증 스크립트 패턴

## 개요

SPEC-B7 기반 슬라이드 중복 검증 자동화 스크립트. Spec 에서 정의된 중복 방지 규칙을 자동으로 검증.

## 사용법

```bash
# 기본 실행
python scripts/validate-slides.py docs/slides.html

# 커스텀 임계값 (기본 80%)
python scripts/validate-slides.py docs/slides.html --threshold 0.75

# 보고서 출력
python scripts/validate-slides.py docs/slides.html --output report.md
```

## 검증 기능

1. **텍스트 추출**: 슬라이드별 본문/다이어그램/팝업 분리
2. **유사도 계산**: Python difflib.SequenceMatcher
3. **중복 감지**: 80% 이상 유사 → 경고
4. **보고서 생성**: Markdown 형식

## SPEC-B7 중복 방지 규칙

### 슬라이드 내 중복 방지
| 위치 | 허용 내용 | 금지 내용 |
|------|-----------|-----------|
| 본문 | 개념/컨셉/개략적 알고리즘 | 다이어그램 요약, 팝업 내용 중복 |
| 다이어그램 | 흐름/구조/순서 | 본문 내용 반복, 텍스트-heavy |
| 팝업 | 기술적 세부사항/구체적 명령어 | 본문 내용 반복, 다이어그램 설명 |

### 슬라이드 간 중복 방지
| 중복 유형 | 규칙 |
|-----------|------|
| 본문 ↔ 본문 | 금지 |
| 다이어그램 ↔ 다이어그램 | 금지 |
| 팝업 ↔ 팝업 | 허용 (공통 개념) |
| 본문 ↔ 다이어그램 | 금지 |
| 본문 ↔ 팝업 | 금지 |
| 다이어그램 ↔ 팝업 | 금지 |

## 출려 예시

```markdown
# 슬라이드 중복 검증 보고서

## 요약
- **총 슬라이드**: 11 개
- **중복 항목**: 0 개
- **검증일시**: 2026-06-04 23:22
- **경고 수준**: LOW

## 중복 목록
| 슬라이드 A | 슬라이드 B | 섹션 | 유사도 |
|-----------|-----------|------|--------|
| — | — | — | 0% |
```

## 구현 참고

- HTML 파싱은 정규식 기반 (간단한 경우만)
- 복잡한 HTML 은 BeautifulSoup 권장
