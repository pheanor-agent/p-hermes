# 이미지 생성 및 관리

Hermes는 내장된 **이미지 생성 스킬(다양한 이미지 모델 등)**을 활용하여 텍스트 프롬프트를 고품질 시각 이미지로 변환할 수 있습니다.

## 🖼️ 이미지 생성 요청하기

Hermes에게 다음과 같이 요청하세요.
> "[TASK] 숲속의 마법 카페를 그려줘. 따뜻한 조명이 특징이야."

에이전트는 해당 요청을 인식하여 최적의 이미지 생성 스킬을 호출하고, 생성된 이미지를 결과로 돌려줍니다.

---

## ⚙️ 이미지 모델 라우팅

에이전트는 작업의 성격에 따라 서로 다른 이미지 생성 모델을 사용합니다.

- **텍스트 렌더링**: 이미지 안에 글자가 포함되어야 할 경우, 텍스트 렌더링이 뛰어난 모델을 사용합니다.
- **실사/예술**: 현실감 있는 사진이나 특정 예술 스타일이 필요할 때 최적화된 모델을 사용합니다.

## 💡 활용 팁
- **상세한 프롬프트**: 조명, 카메라 앵글, 분위기 등 세부적인 정보를 제공할수록 원하는 결과물에 가까워집니다.
- **반복 수정**: "조명을 더 밝게 해줘", "배경을 바꾸줘"라고 요청하면 에이전트가 이전 이미지를 기반으로 수정합니다.
