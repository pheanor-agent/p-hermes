Lecture 04 · Hermes Core Architecture
Q&A
Q. "Memory와 Knowledge의 차이가 뭔가요?"
A. Memory는 '무엇'을 저장하고, Knowledge는 '왜'와 '어떻게'를 이해합니다. 같은 데이터도 층위에 따라 다른 정보가 됩니다.
Q. "Runtime은 어디에 위치하나요?"
A. 4개 레이어 위에서 동작하는 실행 환경입니다. 각 레이어의 상태를 관리하고 외부와의 통신을 담당합니다.
Q. "이 구조를 직접 만들려면?"
A. Memory(DB) → Knowledge(Wiki) → Skills(스크립트) → Workflow(상태 기계) 순서로 구축하세요. L10에서 자세히 다룹니다.
