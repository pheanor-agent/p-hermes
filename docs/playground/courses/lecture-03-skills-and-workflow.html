Knowledge vs Skills
🧠 Knowledge ⚡ Skills
저장 (Store)실행 (Execute)
동적 (Dynamic)확장 (Extensible)
참조 (Reference)적용 (Apply)
"알고 있다""할 수 있다"
무엇을 알아야 하는가무엇을 할 수 있는가
저장과 실행의 차이가 생산성을 결정한다
예시:
"3월 매출 1억 2천"이라는 같은 데이터라도
🧠 Memory: 저장 → 📚 Knowledge: "전월 대비 15% 증가" → ⚡ Skills: 리포트 생성 실행
→ 같은 데이터, 다른 층위, 다른 활용
