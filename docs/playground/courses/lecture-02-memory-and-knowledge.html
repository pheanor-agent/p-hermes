Lecture 02 · Memory & Knowledge
Takeaways
🧠 Memory ≠ Knowledge
기억(사실 저장)과 지식(구조화된 이해)는 근본적으로 다르다
📈 Memory만으로는 부족하다
관리, 분류, 추출 없이 단순 저장만으로 Agent는 성장할 수 없다
🔄 Memory → Knowledge
기억을 조직화하고 연결하는 과정이 Agent OS의 핵심 설계 원칙이다
