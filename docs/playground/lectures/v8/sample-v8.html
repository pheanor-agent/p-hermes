Lecture A Memory: 기억의 힘

🧠

Memory — Agent의 기억

인간이 어제 일을 기억하듯, Agent도 과거를 기억해야 합니다

👤

User Memory — 사용자 기억

당신의 이름, 선호하는 스타일, 자주 사용하는 도구. 마치 개인 비서가 당신을 알고 있듯이, Agent도 사용자에 대해 기억합니다. "김팀장님께서는 항상 간결한 요약을 선호하십니다"

💬

Session Memory — 대화 기록

지금까지 어떤 대화를 나눴는지. "아까 말씀드린 것처럼..." — 이 '아까'가 가능하게 합니다. FTS5 검색으로 수백 세션 중 관련 대사를 찾아냅니다.

📁

Workspace Memory — 작업 기록

어떤 파일을 수정했는지, 어떤 결정을 내렸는지. 마치 작업 일지처럼 프로젝트 진행 상황을 추적합니다. 다음 세션에서도 이어서 작업할 수 있습니다.

💡 비유: Memory는 Agent의 '개인 일기'입니다. 매일 새로운 일기를 쓰면, 과거의 내가 무엇을 했는지 잊지 않습니다.
Memory 3가지 유형 — 탭을 클릭하면 유형별 설명이 전환됩니다
