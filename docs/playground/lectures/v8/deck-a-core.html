Memory — 기억

Agent가 당신을 기억하는 방법

Memory 10 / 34

어제 대화한 내용, 기억하시나요?

Before

기억 없는 Agent

• 매번 소개부터 시작
• 어제 결정 잊어버림
• 같은 질문 반복 응답

After

Memory 있는 Agent

• 이전 대화 맥락 유지
• 사용자 선호도 기억
• 누적된 경험 활용

Memory 11 / 34

두 종류의 기억

💭
세션 메모리
현재 대화 중 기억
= 당신이 지금 생각하는 것
📝
지속 메모리
세션 간 저장/불러오기
= 일기에 적어두는 것
Memory 12 / 34

기억은 어떻게 저장될까요?

📋
1. 감지 → 대화에서 중요한 정보 포착
2. 분류 → 사용자 정보 / 환경 정보 구분
3. 저장 → SQLite 데이터베이스에 기록
4. 활용 → 다음 대화에서 자동 로딩

Knowledge — 지식

Agent가 정보를 체계적으로 다루는 방법

Knowledge 14 / 34

정보는 있지만, 지식이 아닙니다

Before

분산된 정보

• 파일 여기저기 흩어짐
• 검색해도 못 찾음
• outdated 정보 혼동

After

체계화된 지식

• 우선순위 분류
• 0.3초 검색 결과
• 자동 갱신 시스템

VS 15 / 34

VS
📚 Knowledge

= 무엇이 중요한지 아는 것
