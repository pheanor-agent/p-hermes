9단계 상태 머신 — 전체도

핵심 개념을 선택해 살펴보세요

Request → Investigation → Design → Review → Approval은 실행 전 불확실성을 줄이는 구간입니다.
Execution → Test → Execution Review는 승인된 설계를 구현하고 검증하는 구간입니다.
Done은 결과·검증·교훈이 남았을 때만 도달합니다.

      핵심 개념을 선택해 살펴보세요

      Approval은 사용자만 결정하며 Agent는 근거와 선택지를 제공합니다.
      검토 가능한 설계·리스크·산출물이 승인 판단의 입력입니다.

      • 설계된 계획을 사용자에게 제시
      • "확인하고 시작할게요" 또는 "수정 요청" 중 하나
      • 승인 전까지 절대 진행 불가 🔒
