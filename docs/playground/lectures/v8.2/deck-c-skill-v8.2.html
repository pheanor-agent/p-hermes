    분리 ↔ 병합 — 적절한 크기

    분리가 필요할 때

    덩어리가 너무 큰 Skill

    • 100줄이 넘는 긴 Skill
    • 여러 작업을 한 파일에
    → 작은 Skill로 분리

    병합이 필요할 때

    너무 많은 작은 Skill

    • 비슷한 Skill이 5개 넘게
    • 서로 중복된 내용
    → 큰 Skill로 통합

    "올바른 크기의 Skill이 가장 강력합니다"
    너무 크면 복잡하고, 너무 작으면 찾기 힘듭니다.
    한 Skill이 하나의 명확한 문제를 해결할 때 가장 효율적입니다
