Lecture B В· Skill Structure
Skill кө¬мЎ° вҖ” Pyramid Principle
кІ°лЎ  лЁјм Җ: SkillмқҖ м •мқҳ(Definition) вҶ’ мЎ°кұҙ(Condition) вҶ’ мӢӨн–ү(Steps) вҶ’ кІ°кіј(Output)мқҳ 4лӢЁкі„ кі„мёө
рҹ“Ӣ Definition Skillмқҳ м •мІҙм„ұ name, description
вң… Condition мӢӨн–ү м „м ң мЎ°кұҙ when: trigger, guard
вҡҷпёҸ Steps мӢӨн–ү м Ҳм°Ё steps: [action, cmd, parse]
рҹ“Ө Output кІ°кіј нҸ¬л§· format: json, text, raw
в–ё н•ҳмң„ лӢЁкі„ м—Ҷмңјл©ҙ мғҒмң„ лӢЁкі„ мһ‘лҸҷ л¶Ҳк°Җ вҖ” bottom-up dependency
4лӢЁкі„ кө¬мЎ°
  • Definition: Skillмқҳ кі мң мӢқлі„ вҖ” name + description (catalog.json)
  • Condition: м–ём ң мӢӨн–үн• м§Җ вҖ” trigger pattern + guard check
  • Steps: м–ҙл–»кІҢ мӢӨн–үн• м§Җ вҖ” action sequence (cmd/parse/handler)
  • Output: кІ°кіј л°ҳнҷҳ нҳ•нғң вҖ” structured вҶ’ LLM н”„лЎ¬н”„нҠёлЎң
рҹ’Ў Skills = Knowledgeмқҳ мӢӨн–үнҷ” вҖ” м•„лҠ” кІғ(Knowledge)мқҙ м•„лӢҢ, мӢӨн–үн•ҳлҠ” кө¬мЎ°(Skills)
