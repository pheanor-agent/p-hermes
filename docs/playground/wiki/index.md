## 🧠 Core Concepts

Hermes의 핵심 개념을 5~10분 안에 이해합니다.

| 개념 | 설명 | 문서 |
|------|------|------|
| **Job** | Hermes의 작업 단위 — 추적 가능한 공식 요청 | |
| **Knowledge** | 에이전트의 장기 기억 — 지식 베이스 | |
| **Skill** | 재사용 가능한 절차적 지식 | |
| **Workflow Gate** | 9단계 상태머신 — 품질 보증 프로세스 | |
| **Project** | 작업과 지식의 컨테이너 | |
