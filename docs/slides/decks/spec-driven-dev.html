Conflict Detection (3단계 검증)

  1. Quantitative — 수치 비교 (분량, 성능, 라인 수)
  2. Structural — 구조 재정의 감지 (폴더, 트랙, 경로)
  3. Domain Overlap — 도메인 중복 확인
