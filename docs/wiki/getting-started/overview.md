# 에르메스 에이전트 개요

## 한 줄 요약

> **💡 한 줄 요약**: 메시지 채널을 통해 작업을 요청하면, 도구 호출과 모델 연동을 통해 결과를 반환하는 오픈소스 자율 AI 에이전트 플랫폼입니다.

## 기본 개념

Hermes Agent는 사용자의 지시에 따라 실제 작업을 수행하는 자율 AI 에이전트 플랫폼입니다. AI 소프트웨어의 진화 과정에서 세 가지 카테고리가 등장했습니다. 첫 번째는 챗봇입니다. 사용자는 프롬프트를 입력하고 텍스트 응답을 받습니다. 두 번째는 IDE 내 AI 어시스턴트입니다. 코드 보정과 코드 생성을 수행합니다. 세 번째는 에이전트입니다. 사용자의 지시에 따라 외부 도구와 연동하여 실제 작업을 완결합니다.

Hermes Agent는 세 번째 카테고리에 속합니다. 에이전트는 단순한 응답 생성을 넘어, 터미널 명령어 실행, 파일 수정, 웹 검색, 이미지 생성 등 실제 환경에서 작업을 완결합니다.

### 핵심 구성 요소

| 구성 요소 | 역할 | 비유 |
|-----------|------|------|
| 에이전트 | 사용자 요청을 수신하고 도구를 호출 | 팀장 |
| 도구 | 터미널, 파일, 웹 검색, 브라우저 등 | 사원 |
| 모델 | 추론 엔진 (20개 이상 제공자) | 뇌 |
| 스킬 | 재사용 가능한 절차적 지식 | 매뉴얼 |
| 메모리 | 세션 간 유지되는 영속적 지식 | 기억 |
| Gateway | 메시지 플랫폼 연동 | 접수 담당 |

### 핵심 구성 요소

에이전트는 6가지 구성 요소로 작동합니다.

**에이전트(Agent)**는 사용자 요청을 수신하고 필요한 도구를 호출합니다.

**도구(Tools)**는 에이전트가 호출 가능한 기능을 의미합니다. 터미널 명령어 실행, 파일 읽기/쓰기/검색, 웹 검색, 브라우저 자동화, 이미지 생성, 메모리 접근 등 25개 이상의 도구가 제공됩니다.

**모델(Model)**은 에이전트의 추론 엔진입니다. OpenRouter, Anthropic, OpenAI, Z.AI, DeepSeek 등 20개 이상의 제공자를 지원합니다. 모델 변경은 `/model <name>` 명령어로 즉시 적용됩니다.

**스킬(Skills)**은 반복 작업의 절차적 지식을 문서화한 재사용 가능한 모듈입니다. 에이전트는 작업에 맞는 스킬을 자동 로드합니다.

**메모리(Memory)**는 세션 간 유지되는 영속적 지식입니다. 사용자 선호도, 환경 설정, 학습된 교훈을 저장합니다.

**Gateway**는 메시지 플랫폼 연동을 담당합니다. Discord, Telegram, CLI를 통해 사용자와 에이전트가 상호작용합니다.

## 문제 상황

과거 AI 시스템은 사용자의 입력에 텍스트 응답을 생성하는 일방향 서비스였습니다. 프롬프트를 입력하고 답변을 받으며 대화할 수 있었지만, 시스템의 경계 안에서만 작동했습니다. 파일을 읽고 수정하거나, 터미널 명령어를 실행하거나, 외부 서비스와 연동하는 능력은 없었습니다.

에이전트 등장으로 이 경계가 무너졌습니다. 에이전트는 사용자의 환경에 영속적으로 상주하며, 작업을 완수하기 위해 필요한 도구를 자율적으로 선택하고 실행합니다. 사용자는 CLI 앞에 앉아 있지 않아도 메시지 플랫폼을 통해 언제든 에이전트와 상호작용할 수 있습니다.

## 활용 예시

### 코드 작성 및 수정
```
사용자: "Python으로 CSV 파일 분석 스크립트 만들어줘"
에이전트: 파일 생성 → 테스트 실행 → 결과 반환
```

### 이미지 생성
```
사용자: "해변 일몰 배경에 고양이 이미지 생성해줘"
에이전트: ComfyUI 호출 → 이미지 생성 → 결과 반환
```

### 자동화 작업
```
사용자: "매일 오전 9시에 서버 상태 확인 리포트 보내줘"
에이전트: cron 등록 → 스케줄 실행 → 리포트 전송
```

### 연구 분석
```
사용자: "LLM 양자화 기술 현황 조사해줘"
에이전트: 웹 검색 → 논문 스캔 → 요약 보고서 작성
```

## FAQ

**Q: Hermes Agent는 어떤 작업을 수행할 수 있나요?**

코드 작성과 수정, 이미지 생성, 연구 분석, 보고서 작성, 자동화 스크립트 실행 등 다양한 작업을 수행합니다.

**Q: 어떤 모델을 사용할 수 있나요?**

OpenRouter, Anthropic, OpenAI, Z.AI, DeepSeek 등 20개 이상의 제공자를 지원합니다. `/model <name>` 명령어로 변경합니다.

**Q: 에이전트는 어떻게 학습하나요?**

GEPA(Gather, Evaluate, Produce, Apply) 사이클을 통해 경험을 구조화하고 스킬로 변환합니다. 축적된 스킬은 재사용됩니다.

**Q: 에이전트와 상호작용할 수 있는 플랫폼은 무엇인가요?**

Discord, Telegram, CLI를 지원합니다. Gateway 서비스 실행 시 15개 이상의 메시지 플랫폼에 연동됩니다.

**Q: Hermes Agent는 무료인가요?**

Hermes Agent는 MIT 라이선스 오픈소스입니다. 모델 제공자의 API 사용료는 별도입니다.
